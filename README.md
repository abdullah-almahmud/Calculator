# Calculator

A simple calculator built using HTML, CSS, and JavaScript. It supports basic arithmetic operations with a clean and responsive user interface.

## Features

- Addition, subtraction, multiplication, division
- Modulus operation
- Decimal number support
- Backspace and clear functions
- Responsive calculator layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
├── index.html
├── style.css
└── script.js
```
